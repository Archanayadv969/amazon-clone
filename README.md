# 🛒 Amazon Clone

A responsive Amazon homepage clone built using **HTML** and **CSS**. This project was created to improve my front-end web development skills by recreating the layout and design of Amazon's homepage.

## 🌐 Live Demo

🔗 **Live Website:**
https://archanayadv969.github.io/amazon-clone/

## 📌 Project Overview

This project is a static clone of the Amazon homepage that focuses on layout, styling, and responsive design using HTML and CSS.

## ✨ Features

* Amazon-style navigation bar
* Search bar with icon
* Hero section
* Product category boxes
* Responsive footer
* Hover effects
* Clean and organized UI

## 🛠️ Technologies Used

* HTML5
* CSS3
* Git
* GitHub
* GitHub Pages

## 🚀 How to Run Locally

1. Clone the repository

```bash
git clone https://github.com/Archanayadv969/amazon-clone.git
```

2. Open the project folder.

3. Open `index.html` in your browser.
